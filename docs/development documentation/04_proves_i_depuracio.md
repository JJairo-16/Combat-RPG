# Proves i depuració

## Ubicació de proves i incidències

Les proves unitàries desenvolupades per a la verificació del sistema es troben al directori següent:

- [tests](./../../rpg/src/test/java/rpgcombat/)

En aquest directori s'inclouen tots els casos de prova definits per validar el comportament del sistema segons els requisits especificats a la fase de verificació.

D'altra banda, el registre detallat de les incidències detectades (errors, comportaments inesperats i millores) es troba a:

- [bugs](./../bugs/)

Aquest directori conté la documentació de cada incidència, incloent la seva detecció, causa probable i la solució aplicada.

Aquesta organització permet separar clarament la validació del sistema (tests) de l'anàlisi i seguiment d'errors (bugs), facilitant el manteniment i la traçabilitat del projecte.
